# bmp2c
This utility converts a 24 bit color bitmap, or a jpeg image to a 565 RGB format byte array that can be used for the ILI9341 240x320 TFT LCD controller.


	$ python bmp2c.py infile.bmp [-r]
	-r: rotate 90 degree
